VibeReady MVP

This ZIP contains a portable Windows x64 build of VibeReady.

How to start:
1. Extract the ZIP.
2. Double-click VibeReady.exe.

This MVP build is a minimal native Windows client skeleton for CHI-16.
It does not install software, create a system service, run in the system tray,
or start a background resident process.

Target platform:
- Windows 10 x64
- Windows 11 x64

Later MVP issues will add language selection, scanning, repair planning,
controlled installation, local verification, and the Ready page.
