# CHI-15 Trilingual Terminology Freeze

## Tool Names

These tool names must remain unchanged in every locale:

| Canonical | English | 简体中文 | 日本語 |
| --- | --- | --- | --- |
| Git | Git | Git | Git |
| Node.js | Node.js | Node.js | Node.js |
| npm | npm | npm | npm |
| VS Code | VS Code | VS Code | VS Code |

## Voice Rules

- Keep `VibeReady`, `Git`, `Node.js`, `npm`, and `VS Code` untranslated.
- Keep `vibe coding` in Latin characters in all locales.
- Use short action labels on buttons.
- Prefer deterministic status text over playful language.
- Do not expose full filesystem paths, usernames, environment variables, project content, secrets, or tokens in UI copy.
